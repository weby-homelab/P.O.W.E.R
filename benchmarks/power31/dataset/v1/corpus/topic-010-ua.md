# Alerting правила Prometheus

Prometheus alerting rules визначають умови для сповіщень. Наприклад, CPU > 80% для 5 хвилин. Alertmanager маршрутизує сповіщення в Slack або email. Використовуйте inhibit_rules для запобігання дублікатам.
