# Prometheus та Grafana моніторинг

Prometheus збирає метрики з Node Exporter, а Grafana візуалізує їх на дашбордах. Alertmanager надсилає сповіщення при перевищенні порогів. Налаштуйте правила для CPU, пам'яті та дисків.
