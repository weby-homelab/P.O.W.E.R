# TLS сертифікати Let's Encrypt

Let's Encrypt надає безкоштовні TLS сертифікати через протокол ACME. Certbot автоматизує отримання та оновлення сертифікатів. Сертифікати дійсні 90 днів, оновлюйте їх через cron.
