# Docker Healthcheck налаштування

Docker HEALTHCHECK перевіряє працездатність контейнера. Налаштуйте інтервал перевірки та кількість ретраїв. Приклад: HEALTHCHECK --interval=30s --retries=3 CMD curl -f http://localhost
